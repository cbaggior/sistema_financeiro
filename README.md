# 💰 Sistema de Controle Financeiro

Aplicação Flask simples para registrar receitas e despesas.

## 🚀 Funcionalidades
- Adicionar receita/despesa
- Listar transações
- Ver saldo total

## 🛠️ Tecnologias
- Python 3
- Flask
- Pytest

## ▶️ Como executar
```bash
pip install -r requirements.txt
python app/main.py
```

## 🧪 Rodar testes
```bash
pytest
```
